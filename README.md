## How to Run

1. **Clone the repository:**
   ```bash
   git clone https://github.com/yourusername/customer-portal.git
   cd customer-portal
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the application:**
   ```bash
   npm run dev
   ```

4. **Build the application for production:**
   ```bash
   npm run build
   ```
